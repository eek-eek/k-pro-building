# v0.3 План 3 — Фронтенд: дизайн-система + редактирование ресурсов (выполнено)

> Реализован инлайн (цельный рестайл одного экрана — дешевле, чем план→субагент).
> Референс: [smeta-design-system.md](../assets/smeta-design-system.md).

**Goal:** Перевести фронтенд на «Смета Design System» и дать редактирование ресурсного состава прямо в таблице сметы.

## Что сделано

### Коммит `5ccc21a` — дизайн-система (база)
- `frontend/index.html`: подключены **Golos Text** + **JetBrains Mono** (Google Fonts) вместо Inter.
- `frontend/styles.css`: полный рестайл на токены — фон `#F4F3EF`, поверхность `#FFFFFF`, текст `#1C2024`, границы `#E7E5DF`, акцент `#2C5BA8`, статусы ok/warn/rejected/draft. Моноширинные числа (`.num`/`.mono`), компоненты: кнопки (primary/accent/tint/danger/sm), поля (число — вправо, моно), таблица с группами-разделами (синий № раздела, подсветка), блок итогов с «ИТОГО с НДС» в синей заливке, статус-бейджи с точкой, чат/настройки/дашборд.

### Коммит `62d4c28` — редактирование ресурсов + бейджи + Word
- `app.js`:
  - `statusBadge()` — бейдж статуса (рассчитана/черновик) на дашборде и в шапке карточки; `№ id · город` моно-подзаголовок.
  - Дашборд: обёртка `.list`, код-колонка `№ id`.
  - Таблица сметы: строки-работы с ресурсами **раскрываются** (`▸/▾`); под ними — ресурсные подстроки с редактируемыми полями: **вид** (material/labor/machine), **название**, **единица**, **расход**, **цена**; производные **объём** и **сумма**. Цены строки-работы — только чтение (выводятся из ресурсов); объём работы редактируем.
  - **`+ ресурс`** (tint) добавляет ресурс, **✕** удаляет; `syncEdits()` собирает полный массив ресурсов; «Сохранить правки» шлёт в существующий `manual-edit` → сервер пересчитывает → новая версия.
  - `previewTotal()` — клиентский предпросмотр суммы строки из ресурсов; `wireTable()`/`rerenderTbody()` — перерисовка тела таблицы при правках без перезагрузки.
  - `exportDocx()` — под каждой работой выводятся её ресурсы (имя, вид, цена/ед, объём, сумма) вложенными строками.
  - Адаптация KZ: ₸, СН РК/ГОСТ.

## Проверка
- `node --check app.js` — OK; ассеты отдаются свежими; нет ссылок на удалённые классы.
- Сквозной HTTP-смоук: правка расхода ресурса (бетон ×2) → цена/ед строки пересчитана из ресурсов (30 600→61 200), grand total вырос, новая версия; «+ ресурс» (1 чел-ч × 5000) → grand total вырос, ещё версия. Это ровно путь фронта при сохранении.
